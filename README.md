# ThinkAsk
A Website for asking and answering questions made with Django Rest Framework and Vue3

Functionalities:
- User authentication and authorization
- Pagination
- Posting questions
- Posting answers
- Likes
- Dynamic rendering thx to Vue

`./manage.py makemigrations`

`./manage.py migrate`

`./manage.py runserver`

`cd frontend/`

`npm run serve`


`go to localhost:8000`
