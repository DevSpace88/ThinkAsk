# ThinkAsk
A Website for asking and answering questions made with Dajngo Rest Framework and Vue3

Functionalities:
- User authentication and authorization
- pagination
- Likes
- posting questions
- posting answers
- dynamic rendering thx to Vue

`./manage.py makemigrations`

`./manage.py migrate`

`./manage.py runserver`

`cd frontend/`

`npm run serve`


`go to localhost:8000`
